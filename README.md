# SK Maletaht - Shahkklubi Veebisait

Tere tulemast SK Maletaht shahkklubi ametlikule veebisaidile!

## 📋 Projekti Kirjeldus

SK Maletaht on Eesti üks vanemaid ja prestižeeritud shahtklusid. See projekt sisaldab modernatele veebisaidi, mille eesmärk on:

- Tutvustada klubit ja selle ajalugu
- Näidata tulevasi turniiride ja sündmusi
- Kuulutada klubide parimaid liikmeid
- Pakkuda kontaktinformatsiooni

## 🎨 Funktsioonid

- **Avaleht** - Atraktiivne sissejuhatus klubiga
- **Meist** - Info klubi kohta ja eripärad
- **Turniirid** - Tulevased turniiriad ja registreerimine
- **Liikmed** - Klubis osaleva parimad shahkijad
- **Kontakt** - Kontaktivorm ja kontaktandmed

## 🛠️ Tehnoloogiad

- HTML5
- CSS3 (responsiivsed disainid)
- JavaScript (interaktiivsus)

## 📱 Responsive Design

Veebisait on optimeeritud kõigile seadmetele:
- Desktop
- Tablet
- Mobiil

## 🚀 Kasutamine

1. Avage `index.html` brauseris
2. Navigeerige erinevate osade vahel
3. Registreeruge turniiridele
4. Saatke sõnumid kontaktivormi kaudu
